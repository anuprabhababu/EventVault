Project Description
Helps the users to organise their events efficiently by allowing users to store details of  events,deadlines, get remainders and upload documents.

The Problem statement
Organising events become a hectic task as the users need to store details of program in one app, set remainder in another and store associated files in another.

The Solution
All these features are integrated into single platform for user convenience
